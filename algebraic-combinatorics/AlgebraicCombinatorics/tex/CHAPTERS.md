# Textbook Chapter Structure

This file documents the structure of the LaTeX source files that were extracted from
Darij Grinberg's textbook *Math 701 — Algebraic Combinatorics*. The `.tex` files have
been removed for copyright reasons.

To recreate them, obtain the relevant sections from the original textbook.
